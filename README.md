# Engagement Management System

## Project Overview
The **Engagement Management System** is a robust solution designed to streamline and monitor business engagements within Salesforce. Leveraging modern development practices, this system integrates **Custom Objects**, **Lightning Web Components (LWC)**, **Apex Controllers**, and **Salesforce Flows** to provide a comprehensive management interface for tracking engagement health and activity.

## Assumptions
To ensure consistent data integrity and expected behavior, the following logic applies:
*   **Currency**: All financial metrics and Opportunity amounts are calculated and displayed in **USD**.
*   **Activity Metrics**: Only Tasks with a Status of **'Completed'** are included in the engagement summary calculations.
*   **Automation**: The core Flow automation is designed to trigger exclusively when an Engagement record reaches the **'Negotiation/Review'** stage.

## Testing Guide
Follow these steps to verify the system's functionality:

### 1. Flow Automation
Navigate to an Engagement record and change its **Stage** to **'Negotiation/Review'**. Verify that the automated Task or action is triggered as defined in the business logic.

### 2. LWC Summary Metrics
View the `Engagement Metrics` card on the record page. Ensure the **Total Opp Amount**, **Completed Tasks**, and **Upcoming Events** reflect real-time data from child records.

### 3. Activity Tracking
Log a Task and mark it as 'Completed'. Observe the LWC metric for "Completed Tasks" incrementing dynamically. Likewise, create a future Event to verify the "Upcoming Events" count.
